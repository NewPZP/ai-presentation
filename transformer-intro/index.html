Attention Is All You Need

30 分钟讲清楚 Transformer 架构核心原理

为什么需要 Transformer？

  • RNN 无法并行，训练慢
  • 长距离依赖信息衰减
  • Attention 机制可以全局建模
  • 2017 年论文：抛弃 RNN/CNN，纯 Attention

Self-Attention 机制

Q = X · Wq (Query: 我在找什么？) K = X · Wk (Key: 我有什么？) V = X · Wv (Value: 我的实际内容) Attention(Q, K, V) = softmax(Q · Kᵀ / √dₖ) · V

每个 token 都能直接"看到"序列中的任意其他 token

Multi-Head Attention

  • 将 Q/K/V 拆成 h 个头
  • 每个头学习不同的注意力模式
  • 类比：多视角观察同一场景
  • 拼接后线性投影 → 最终表示

整体架构

┌─────────────────────┐ │ Output Embedding │ ├─────────────────────┤ │ Decoder × N layers │ │ ┌─ Masked Attn │ │ ├─ Cross Attn │ │ └─ FFN │ ├─────────────────────┤ │ Encoder × N layers │ │ ┌─ Self Attn │ │ └─ FFN │ ├─────────────────────┤ │ Input Embedding │ └─────────────────────┘

关键要点

  • Self-Attention 是核心，替代了循环和卷积
  • Positional Encoding 补偿位置信息
  • Layer Norm + Residual 保证训练稳定
  • 可扩展：GPT / BERT / ViT 都是变体
